显示所有区生僻字(vscode, 同文输入法, 小狼毫输入法)

所需字体文件

Plangothic 字体
Noto Sans 字体
全宋体 字体

安装

win11安装 PlangothicP1-Regular.allideo.ttf, PlangothicP2-Regular.ttf, NotoSansSC-Regular.ttf 字体文件.

  • 这几种字体与微软雅黑字体类似, 显示不会太凸兀.全宋体文件中有个简易测试文本.
  • NotoSansSC是安卓默认字体
  • 这几种字体可以包括汉字基本区, 扩展A, B, C, D, E, F, G, H, I 区, 相容表意区, 相容表意补充区.

VS code

文件-首选项-字体-Editor: Font Family

首选 Microsoft YaHei, 其它 Plangothic P1, Plangothic P2, Noto Sans SC 为备选字体.
