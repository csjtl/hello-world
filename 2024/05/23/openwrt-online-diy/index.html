OpenWrt在线编译（官方网址）及自定义（最适合我这种懒人）

发表于 2024-05-23 分类于 embedded system ， openwrt


现在不是很想折腾了，稳定在了OpenWrt 官方的在线编译了。想升级时将参数拷贝进去，直接点击构建。

硬件配置

主板：h77n-wifi （老主板，设备刷写部分有BIOS添加nvme模块实现pcie启动系统）

cpu：i5-3470t

硬盘：英特尔傲腾16G的

在线编译

编译网址：https://firmware-selector.openwrt.org/

本人常用型号：x86和ac88u；型号右侧有openwrt版号。

软件包

点击 ‘自定义已安装的软件包和/或首次启动脚本’ 三角展开。已安装的软件包中可以添加你需要的插件需要的依赖包名（官方源没有的插件）。

这是我自用的x86的包，包含了GitHub中插件oc所需的所有依赖。安装完成只需要把ipk文件上传安装一下就成了。

启动脚本

uci-defaults启动可以实现各种自定义

这是我目前自用的

cat << "EOF" > /etc/uci-defaults/99-custom 创建名为99-custom的启动文件

set system.@system[0].hostname='OpenWrt-X86' set system.@system[0].zonename='Asia/Shanghai' 主机名和时区

set ttyd.@ttyd[0].command='/bin/login -f root'ttyd自动登录

set network.lan.ipaddr='10.0.0.1' set network.lan.netmask='255.255.255.0' set network.wan=interface set network.wan.device='eth1' set network.wan.proto='pppoe' set network.wan.username='XXXXXXXXXX' set network.wan.password='mmmmmmm'lan口ip和拨号相关

set ddns.ddns=service set ddns.ddns.service_name='google.com' set ddns.ddns.lookup_host='xxxxx' set ddns.ddns.domain='xxxxx' set ddns.ddns.username='xxxxx' set ddns.ddns.password='xxxxx' set ddns.ddns.ip_source='interface' set ddns.ddns.interface='pppoe-wan' set ddns.ddns.ip_interface='pppoe-wan'ddns相关

passwd root << EOI 8888 8888 EOI设置root密码为8888

其它可使用uci set的自定参数可在自己已在运行使用的 openwrt 的终端或 ssh 中使用uci show命令显示。这些参数可以添加到uci-defaults中修改。

设备刷写

我是u盘安装Ventoy，直接加载pe的iso文件，在pe中使用 IMG写盘工具.exe 直接写入ssd。

因为是老主板，要pcie启动ssd 需要添加nvme模块，有兴趣可以查看一个 pcie 插槽损坏的 h77n-wifi（BIOS添加nvme模块实现pcie启动操作系统）这篇文章。

进入pe后打开 IMG写盘工具 程序，选择要写入的盘和需要写入到盘的img文件。

开始刷写

完成后重启退出pe，进入op。

这次重启进入op后uci-defaults中的文件并没有执行，因为文件还存在，op执行完成后会删除uci-defaults中文件。

所以，reboot一下就行了。
