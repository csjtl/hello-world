pi4 外壳（ argon one 按键开关机，上壳金属）

发表于 2020-05-10 更新于 2021-10-24 分类于 embedded system ， raspberry pi

argon one pi4外壳：对这个产生兴趣是在看广告时，发现它有一个电源按键可以实现开关机。（pi4也是可以通过gpio引脚实现开关机，可以参考/boot/overlays/README文档）。对它的设计产生好奇，就买了一个一探究竟。

部件爆炸图

几个大件，加少量螺丝。

上壳有三部分组成：金属外壳，散热风扇，电源按键扩展。

电源扩展：供电经过电子元件通过gpio给pi4供电，电源旁边有个按键。

耳机孔，mini-hdmi接口扩展板。这个板没有啥，线路物理直连。

直接插进去就行。

在上壳贴上导热片扣上pi4就行。

网卡出工艺处理粗糙，用锉磨了磨才正好装进去。

组装完成，后部这样子的。
