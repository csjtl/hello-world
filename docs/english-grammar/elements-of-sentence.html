句子的要素

造句的两个条件：

  1. 一个句子的组成要素：
    主语，谓语，宾语，表语，补语，定语，状语，不常用的定位语。
  2. 组成成份的位置和顺序。

五个基本句型

表示状态

状态型句子（主语 + be（系动词）+ 表语 + 状语）

  • 主语 + be（系动词）+ 表语 + 状语

表示动作

动作型句子（主语 + 谓语（实义动词） + 宾语 + 状语）

  • 主语 + 谓语（不及物动词）+ 状语
  • 主语 + 谓语（及物动词）+ 宾语 + 状语
  • 主语 + 谓语（及物动词里面的双宾动词）+ 双宾语 + 状语
  • 主语 + 谓语（少数特定的及物动词）+ 宾语 + 宾补 + 状语

实义动词（及物动词和不及物动词）。

要素

主语：

主语是一个句子叙述的主体。
可以作为主语的词：名词·代词·数词·动词不定式·动名词·句子·其它。
不可以作主语的词：谓语动词，形容词，介词，副词等。

  1. 名词
    名词就是世间万物的名称。
    名词都可以作为主语。

  2. 代词
    代词就是为了避免重复而代替名词的词语。
    绝大多数代词可以作为主语（ 人称代词的主格可以作为主语，宾格不能作为主语）。

  3. 数词
    数词有基数词（1，2，3…）和序数词（第一，第二，第三…）。
    不管基数词还是叙述词都可以作为主语。

  4. 动词不定式或短语
    动词不定式和动名词统称：非谓语动词。
    表示动作的词语作主语时，要变成非谓语形式，也就是动词不定式或者动名词

  5. 句子
    作为主语的句子叫做主语从句。

  6. 其它
    有些词语看起来不是名词，但具有名词的含义，可以作主语。

谓语

谓语定义：说明主语的动作或者状态的词语。
谓语有动作型谓语和状态型谓语两种：动作性谓语和状态型谓语。

  1. 动作型谓语：由动词构成的，动词就是表示动作或心理行为的词语。
    动作型谓语又分为简单谓语和复合谓语。

    • 简单谓语：实义动词或动词短语构成。
      • 实义动词（有实际意义的动词）。
      • 动词短语由动词和一个单词或几个单词构成的词组。因为几个词共同表示一个动作，所以动词短语也称为复合动词。
    • 复合谓语
      • 复合谓语：简单谓语加上其它词语，共同作为谓语。
      • 三种表示形式：
        1. 情态动词 + 动词原形（实义动词或动词短语原型）
        2. 助动动词 + 动词原形（实义动词或动词短语原型）
        3. 情态动词 + 动词的其他格式实义动词（be + doing 或 Have/has + done）

  2. 状态型谓语：由系动词 + 表语构成。

    • 系动词 + 表语也是一种复合谓语。

表语

  • 表语属于谓语的范畴，但不是谓语，系动词 + 表语才是谓语。系表结构不能分开，两者同时存在。
  • 可以做表语的词：名词，形容词，介词短语，名词所有格，动词不定式，动名词，代词，数词，分词，副词，句子都可以作表语。

宾语

  • 宾语：动作的对象或者承受者。实义动词分为及物动词和不及物动词。

  • 及物动词后面一定要跟一个动作的对象或者承受者（宾语）。

  • 有些及物动词后面可以跟两个宾语：并列宾语或双宾（直接宾语，间接宾语）。

    • 直接宾语：一人一物。

  • 能带两个宾语的及物动词称为双宾动词。

  • 能作主语（名词·代词·数词·动词不定式·动名词·句子·其它）的词语都可以作宾语。

    • 两处不同之处：
      1. 代词：可以作主语，也可以作宾语。人称代词主格，只能作主语，人称代词宾格只能作宾语。
      2. 句子：可以作主语，也可以作宾语。名称不一样，句子作主语叫主语从句，句子作宾语叫宾语从句。

  • 其它能作宾语的词语：动词不定式，动名词，句子等。

  • 不能作主语的词语，也不能做定语。

宾语补足语（宾补）

  • 特定的及物动词加上宾语后意思还是不完整，需要加上一个词语补充宾语，使句子意思完整。这个补充宾语的词语叫做宾补。

  • 宾补只跟宾语有关系。

  • 能作宾补的词语：形容词，名词，数词，动词不定式，动名词，副词，分词，介词短语，句子。

    • 形容词
      主语 + 特定的及物动词 + 宾语 + 宾补
      主语 + make/keep + 宾语 + 形容词
    • 名词
      主语 + make/call + 宾语 + 名词（宾补）
      主语 + make/call + 宾语 + 名词（宾语）
      名词作宾补的时候容易跟双宾混淆。
      • 宾补：宾语 + 名词都表示同一人或同一事物
      • 双宾：宾语 + 名词都表示一人一物
    • 数词
    • 动词不定式
    • 动名词
    • 副词
    • 分词
    • 介词短语
    • 句子

状语

  • 状语是一个很重要的句子成分。

  • 状语：说明谓语的情况，状语只跟谓语有关系。

  • 状语一般放在句子的后面。

  • 状语种类：时间状语，地点状语，方式状语，原因状语，目的状语等等。

  • 状语一般由副词，介词短语充当。

  • 地点状语可以修饰动作性谓语（主语 + 系动词 + 表语 + 状语），也可以修饰状态型谓语。

  • 时间副词可以作时间状语。
    方式副词可以作方式状语。
    频度副词可以做时间状语。

  • 一句英文句子中，可以同时出现多个状语。

定语

  • 定语：修饰或限定名词的词语。名词都是需要定语（共同存在，特殊情况带）。修饰不定代词的词语也叫定语（放不定代词后面，属于后置定语），

  • 普通名词才能带定语，专有名词不能带定语（特殊情况带）。

  • 定语分为：前置定语（定语在名词前面）和后置定语（定语在名词后面）。

    • 可以作前置定语：形容词性物主代词，形容词，名词所有格（`s），数词，名词，量词，指示代词等等。
    • 同一个普通名词可以跟多个定语。
    • 后置定语构成：介词短语，名词所有格（of + 名词），动名词短语，动词不定式短语，句子构成。

同位语

  • 同位语：两个不同的词语都表示同一个人或事物。起强调补充作用。
