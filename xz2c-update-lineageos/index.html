索尼 XZ2C LineageOS 20 更新 21

发表于 2024-05-29 更新于 2024-05-30 分类于 system ， phone


索尼 XZ2C LineageOS 20 更新 21

需要准备的

LineageOS 21 安装包
Google Android 14 或 Google 套装
Magisk Delta 文件
adb 驱动文件
fastboot 驱动安装（看文章内容）
USB 驱动

安装驱动

adb

  • 解压 platform-tools-latest-windows.zip 压缩包到指定目录，并添加系统变量。

  • shell 或 cmd
    输入 adb devices 验证是否识别到设备
    输入 adb -d reboot sideload 重启设备到 Recovery 模式

fastboot

在 Recovery 模式查看设备管理器中设备驱动，有感叹号是没有驱动

安装：

  1. 感叹号设备上右键-更新驱动程序
  2. 浏览我的电脑以查找驱动程序
  3. 从计算机可用驱动程序列表选取
  4. 硬件类型找到 Android Phone
  5. 型号选择 Android Bootloader Interface 确认安装

输入 fastboot devices 验证是否识别到设备，不能识别没有输出。

usb（没有用到）

升级刷写

官方流程

Recovery 模式

输入 adb -d reboot sideload 重启设备到 Recovery 模式

刷写系统

输入 adb -d sideload .\lineage-21.0-20240524-nightly-xz2c-signed.zip 刷写系统
lineage-21.0-20240524-nightly-xz2c-signed.zip 文件在当前文件夹内

注：系统刷写有个说明

不需要全家桶

直接完成刷机重启

需要全家桶

不能重启进系统，要选择 Advanced ，选择 Reboot to Recovery ，重启到 Recovery 模式后，选择 Apply Update ，选择 Apply from ADB，在执行下面的刷写。

Google 全家桶

输入 adb -d sideload .\MindTheGapps-14.0.0-arm64-20240225_232108.zip 刷写 Google 全家桶

Magisk Delta

输入 adb -d sideload .\26.4-kitsune.zip 刷写面具 apk。zip 文件是面具 apk 文件直接修改文件名来的。

完成重启
